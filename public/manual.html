07 — BOM ANALYSIS

구매 자료 즉시 완성

🔩

📏

AUTO CALC
📦
PROCUREMENT
타입별 총 필요 길이 + 5% 여유율 + 중량(kg) 자동 합산. 구매 발주서 즉시 생성.
PROCUREMENT
🧱
COAMING BOM
COMPOUND / MANGANA 자동 산출. 충전율 35% 기준 체적·중량 계산.
SEALING
🏷️
NAME TAG BOM
FROM/TO 2개 + CTYPE 코밍 토글 지원. 케이블 수량에 맞는 명판 발주 수 자동 계산.
